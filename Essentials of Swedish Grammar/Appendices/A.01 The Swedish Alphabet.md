# A.1 The Swedish Alphabet

## Source Correspondence

This appendix lists the Swedish alphabet. Swedish uses the Latin alphabet plus the letters `å`, `ä` and `ö`.

## Alphabet

| Letter | Note |
|---|---|
| A a | Swedish vowel |
| B b | consonant |
| C c | mostly in loanwords and names |
| D d | consonant |
| E e | Swedish vowel |
| F f | consonant |
| G g | consonant |
| H h | consonant |
| I i | Swedish vowel |
| J j | consonant |
| K k | consonant |
| L l | consonant |
| M m | consonant |
| N n | consonant |
| O o | Swedish vowel |
| P p | consonant |
| Q q | rare, mostly in names and loanwords |
| R r | consonant |
| S s | consonant |
| T t | consonant |
| U u | Swedish vowel |
| V v | consonant |
| W w | mostly in names and loanwords |
| X x | consonant letter |
| Y y | Swedish vowel |
| Z z | mostly in names and loanwords |
| Å å | Swedish vowel; separate letter after z |
| Ä ä | Swedish vowel; separate letter after å |
| Ö ö | Swedish vowel; separate letter after ä |

## Alphabetical Order

The final three Swedish letters come after `z`:

```text
... x, y, z, å, ä, ö
```

## Study Notes / Summary

### 中文总结

瑞典语字母表包含 `å`、`ä`、`ö`，它们不是 a/o 的装饰形式，而是独立字母，排序在 `z` 之后。
