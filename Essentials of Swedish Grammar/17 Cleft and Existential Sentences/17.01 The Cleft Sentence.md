# 17.1 The Cleft Sentence

## Source Correspondence

This section explains the cleft sentence, used to put special focus on one part of the sentence.

## Pattern

```text
Det är / Det var + focused element + som + clause
```

| Neutral Sentence | Cleft Sentence | Focus |
|---|---|---|
| Anna ringde i går. | Det var Anna som ringde i går. | Anna |
| Anna ringde i går. | Det var i går som Anna ringde. | yesterday |
| Jag träffade Erik. | Det var Erik som jag träffade. | Erik |

## Function

A cleft sentence presents one part as especially important, contrastive or new.

| Use | Example |
|---|---|
| identify a person | Det är Maria som har nyckeln. |
| identify a time | Det var på måndag som mötet började. |
| identify an object | Det är den här boken som jag behöver. |

## Study Notes / Summary

### 中文总结

Cleft sentence 用 `Det är/var X som...` 突出 X。它常用于纠正、强调或识别某个信息点。
