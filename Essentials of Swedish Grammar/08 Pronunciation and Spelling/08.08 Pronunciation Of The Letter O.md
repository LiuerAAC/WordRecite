# 8.8 Pronunciation Of The Letter O

## Source Correspondence

This section explains why the letter `o` is difficult in Swedish: it can represent the `o` sound, but in some words it represents an `å` sound.

## Main Patterns

| Letter | Sound | Example | Note |
|---|---|---|---|
| o | o | sol | normal long `o` |
| o | å | lov | exception |
| o | short o | ost | relatively few words |
| o | short å | jobb | common short `å` spelling |

The letter `å` itself is always pronounced as an `å` sound.

## Compare

| Word | Sound | Meaning |
|---|---|---|
| hov | o | hoof |
| hov | å | court |
| håv | å | bag net |
| kol | å | coal |
| kål | å | cabbage |
| kort | o | card |
| kort | å | short |

## Study Notes / Summary

### 中文总结

字母 `o` 可能读作 `o`，也可能读作 `å`，尤其短 `å` 常写作 `o`。字母 `å` 本身不会造成问题，总读 `å` 音。

### 检查点

- 是否知道 `o` 不是总读 o？
- 是否能记住 `jobb` 是短 `å` 音？
- 是否能区分 `kort` 的两个读音/意义？
