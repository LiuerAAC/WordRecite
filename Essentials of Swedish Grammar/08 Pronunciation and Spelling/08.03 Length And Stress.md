# 8.3 Length And Stress

## Source Correspondence

This section explains Swedish vowel length and stress. Swedish distinguishes long and short vowels in pronunciation. Ordinary spelling does not consistently show vowel length, so the book uses special marks in pronunciation sections.

## Length Marking

| Normal Text | Marked Text | English |
|---|---|---|
| tal | tal | speech |
| tall | tall | pine |
| rik | rik | rich |
| risk | risk | risk |

Only the stressed vowel is marked. An unmarked vowel is unstressed and short.

## Stress Patterns

| Stress On First Vowel | English |
|---|---|
| vila | rest |
| villa | house |
| silar | strainers |
| sillar | herrings |

| Stress Not On First Vowel | English |
|---|---|
| magasin | warehouse |
| tablett | tablet |
| teater | theatre |
| paket | packet |
| metall | metal |
| betona | stress |
| försök | try |
| behålla | keep |

## Study Notes / Summary

### 中文总结

瑞典语元音有长短区别，重音通常落在一个元音上。长短标记只标在重读元音下；未标记元音通常为非重读短元音。

### 检查点

- 是否知道 vowel length 和 stress 要一起看？
- 是否知道一个词通常只有一个重读元音？
- 是否能比较 `vila` 与 `villa`？
