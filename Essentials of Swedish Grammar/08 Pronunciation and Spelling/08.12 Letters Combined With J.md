# 8.12 Letters Combined With J

## Source Correspondence

This section lists letter combinations with `j` that are pronounced as one sound, plus combinations where the first consonant is silent and the whole combination is pronounced like `j`.

## Common Examples

| Spelling | Example | English |
|---|---|---|
| dj | djup | deep |
| gj | gjorde | did |
| hj | hjälpa | help |
| lj | ljus | light |
| sj | sjuk | sick |
| stj | stjärna | star |
| skj | skjorta | shirt |
| tj | tjock | fat |
| kj | kjol | skirt |

## Loan-Word Spellings

| Sound | Spellings / Examples |
|---|---|
| sj | schack, schema, sherry, shoppa, chock, chef |
| tj | check, charter |
| -sjon | station, lektion, diskussion |

## Study Notes / Summary

### 中文总结

很多含 `j` 的拼写组合读作一个音，如 `sj`, `tj`, `hj`, `lj` 等。外来词中 `sch/sh/ch` 也可能表示 `sj` 或 `tj`。

### 检查点

- 是否能认出 `sjuk`, `skjorta`, `stjärna` 的共同读音？
- 是否知道 `hjälpa/ljus` 中首辅音不单独发？
- 是否知道 `-tion/-sion` 常读作 `-sjon`？
