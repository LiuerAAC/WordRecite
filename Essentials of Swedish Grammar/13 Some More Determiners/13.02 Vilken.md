# 13.2 Vilken

## Source Correspondence

This section introduces `vilken`, a determiner used in questions and exclamations. It corresponds to English which or what.

## Forms

| Noun Type | Form | Example | English |
|---|---|---|---|
| `en` word | vilken | Vilken bok? | Which book? |
| `ett` word | vilket | Vilket hus? | Which house? |
| plural | vilka | Vilka böcker? | Which books? |

## Use

`vilken` asks for identification from a set or asks what kind of noun is meant.

| Swedish | English |
|---|---|
| Vilken buss går till centrum? | Which bus goes downtown? |
| Vilket språk talar du? | What language do you speak? |
| Vilka filmer tycker du om? | Which films do you like? |

## Study Notes / Summary

### 中文总结

`vilken/vilket/vilka` 根据后面的名词变化，表示“哪一个、什么样的”。`vilket språk` 用 `vilket`，因为 `språk` 是 ett-word。

### 检查点

- `vilken` 如何对应 `en/ett/plural`？
- `Vilket språk?` 为什么不用 `vilken`？
- `vilka` 用在什么名词前？
