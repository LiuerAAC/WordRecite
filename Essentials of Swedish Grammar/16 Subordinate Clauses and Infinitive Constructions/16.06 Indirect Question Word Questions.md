# 16.6 Indirect Question-Word Questions

## Source Correspondence

This section explains indirect questions introduced by a question word.

## Pattern

```text
main clause + question word + subordinate clause
```

| Direct Question | Indirect Question |
|---|---|
| Var bor hon? | Jag vet inte var hon bor. |
| Vad gör du? | Han frågar vad du gör. |
| När kommer tåget? | Vet du när tåget kommer? |
| Varför skrattar de? | Jag undrar varför de skrattar. |

## Question Words

| Swedish | English |
|---|---|
| vem | who |
| vad | what |
| var | where |
| vart | where to |
| när | when |
| varför | why |
| hur | how |

## Study Notes / Summary

### 中文总结

带疑问词的 indirect question 保留疑问词，但后面用从句词序：`var hon bor`，不是 `var bor hon`。

### 检查点

- `Var bor hon?` 嵌入后为什么变成 `var hon bor`？
- `vart` 和 `var` 的差别是否仍然保留？
- indirect question 中疑问词是否消失？
