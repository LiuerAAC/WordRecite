# 16.3 Important Verbs Followed By An Att Clause Or The Infinitive As Object

## Source Correspondence

This section groups verbs according to whether they are followed by an infinitive phrase or an `att` clause.

## Infinitive Complements

| Verb | Pattern | Example |
|---|---|---|
| vilja | vilja + infinitive | Jag vill resa. |
| kunna | kunna + infinitive | Hon kan simma. |
| försöka | försöka + infinitive | Vi försöker förstå. |
| börja | börja + infinitive | De börjar arbeta. |

## `Att` Clause Complements

| Verb | Pattern | Example |
|---|---|---|
| tro | tro att + clause | Jag tror att han kommer. |
| säga | säga att + clause | Hon säger att hon är sjuk. |
| veta | veta att + clause | Vi vet att det är sant. |
| hoppas | hoppas att + clause | De hoppas att vädret blir bättre. |

## Learning Note

Some English verbs map naturally to an infinitive in Swedish; others introduce a full clause with `att`. Learn them as verb patterns.

## Study Notes / Summary

### 中文总结

动词后接什么补语要按搭配记。`vill/kan/försöker` 常接 infinitive；`tror/säger/vet/hoppas` 常接 `att` 从句。

### 检查点

- `Jag vill resa` 为什么没有 `att`？
- `Jag tror att...` 后面是什么结构？
- 为什么这类动词最好按 pattern 记？
